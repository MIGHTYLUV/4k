3. Add your raw manifest link:
```text
https://raw.githubusercontent.com/<YOUR_GITHUB_USERNAME>/<YOUR_REPO_NAME>/main/manifest.json
```
4. Enable the scrapers and enjoy strictly 4K & premium 1080p streaming!
